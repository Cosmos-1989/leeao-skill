# 李敖.Skill

**一个面向 Codex 的本地分析知识库仓库**

一个面向 Codex 的本地分析知识库仓库，目标是复用李敖常见的论战风格、论证习惯和材料组织方式。

### 设计原则

- **蒸馏方法，不复制全集**
- 学论证锋芒，不只学骂人
- 引用必须核对原文
- 对争议性表达保留边界

### 快速开始

```bash
git clone https://github.com/Cosmos-1989/leeao-skill
cd leeao-skill
```

然后在 Codex 中打开该仓库，直接提问即可。

### 适用场景

适合：分析人物、整理材料、风格改写、论战参考

不适合：COPY式模仿、情绪化表达、原样照搬

### 仓库结构

```
leeao-skill/
├── AGENTS.md
├── SOUL.md
├── TOOLS.md
├── skills/leeao/SKILL.md
├── prompts/
├── knowledge/
├── docs/
└── facts/
```

### 工作流

1. 先读 `AGENTS.md` 规定读取顺序和工作边界
2. 读 `TOOLS.md`、`SOUL.md` 建立行为框架
3. 读 `skills/leeao/SKILL.md` 了解具体 skill
4. 读 `prompts/literary_style.md` 获得风格参考
5. 读 `knowledge/quickstart.md` 快速入门
